# 🧠 Stroke Risk Prediction System

A Machine Learning-based web application that predicts the risk of stroke using patient health parameters. The system provides an easy-to-use interface for healthcare risk assessment by analyzing patient data and generating accurate predictions using trained machine learning models.

---

## 📖 Overview

Stroke is one of the leading causes of death and long-term disability worldwide. Early prediction of stroke risk can help individuals take preventive measures and seek timely medical attention.

This project uses Machine Learning algorithms to analyze patient health information and estimate the probability of stroke. The application is built with Flask, providing a simple and interactive web interface for users.

---

## 🎯 Objectives

- Predict stroke risk using Machine Learning models.
- Provide a user-friendly web application for health risk assessment.
- Compare multiple machine learning algorithms to improve prediction accuracy.
- Generate prediction reports for users.
- Maintain prediction history for future reference.

## 🤖 Machine Learning Models

The project evaluates multiple machine learning algorithms to identify the most suitable model for stroke risk prediction.

- Random Forest Classifier
- Logistic Regression
- K-Nearest Neighbors (KNN)

Among these, Random Forest is used as the primary prediction model due to its strong performance and reliability.

## ⚙ Installation

### Clone the Repository

```bash
git clone https://github.com/yourusername/Stroke-Risk-Prediction.git
```

### Navigate to the Project

```bash
cd Stroke-Risk-Prediction
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
cd app
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
